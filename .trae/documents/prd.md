## 1. 产品概述

iPhone 换屏配件匹配工具，面向维修门店前台、配件采购员和新手技师，极简快速查询界面，打开就能查。解决维修行业屏幕选型复杂、兼容性不明确、价格不透明的痛点，提高门店运营效率。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 使用场景 | 核心需求 |
|------|----------|----------|
| 门店前台 | 客户接待、快速报价 | 快速匹配型号、生成对比方案、历史报价查询 |
| 配件采购员 | 备货采购、库存管理 | 价格对比、毛利计算、批次备注 |
| 新手技师 | 维修操作、兼容性确认 | 兼容提醒、测试清单、常见风险提示 |

### 2.2 功能模块

1. **型号选择页**：iPhone 型号选择、国行/外版、Face ID 状态、原屏类型、客户预算
2. **屏幕方案页**：可用屏幕等级列表（原拆、后压、国产 OLED、LCD 替代）、参数对比、风险提示
3. **兼容提醒页**：听筒排线、原彩写入、密封胶、支架、螺丝规格等注意事项
4. **库存报价页**：现有数量、进货价、建议零售价、毛利排序
5. **对比清单页**：勾选 2-3 种方案生成客户解释卡

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 型号选择页 | 型号筛选器 | 支持 iPhone 6 到 iPhone 15 Pro Max 全系列选择，按发布时间倒序排列，支持常用机型置顶 |
| 型号选择页 | 参数选择区 | 国行/外版切换、Face ID 正常/异常、原屏类型（OLED/LCD）、预算滑块（0-3000 元） |
| 屏幕方案页 | 方案卡片列表 | 展示原拆、后压、国产 OLED、LCD 替代四种等级，卡片包含亮度、色彩、厚度、触控手感、原彩支持、常见风险 |
| 屏幕方案页 | 参数对比条 | 每个方案用进度条可视化显示各项参数得分，一目了然 |
| 兼容提醒页 | 兼容性列表 | 听筒排线兼容性、原彩写入要求、密封胶型号、支架类型、螺丝规格对照表 |
| 兼容提醒页 | 异常案例收藏 | 可收藏特殊兼容问题案例，方便后续查阅 |
| 库存报价页 | 报价表格 | 按现有库存数量、进货价、零售价、毛利多维度排序，支持快速筛选 |
| 库存报价页 | 批次备注 | 记录配件批次信息、供应商、采购日期、质保期限 |
| 对比清单页 | 方案勾选器 | 支持 2-3 种方案勾选，生成可视化对比解释卡 |
| 对比清单页 | 客户解释卡 | 简洁明了的对比表格，包含价格、质保、优缺点、推荐理由 |
| 全局功能 | 换屏后测试清单 | 触控、显示、原彩、Face ID、感光、听筒等功能测试项 |
| 全局功能 | 历史报价查询 | 按日期、型号、客户查询历史报价记录 |

## 3. 核心流程

用户选择 iPhone 型号及参数 → 系统自动匹配合规的屏幕方案 → 查看兼容性注意事项 → 浏览库存报价 → 勾选方案生成对比清单 → 完成报价

## 4. 用户界面设计

### 4.1 设计风格

**极简工业风**，面向专业维修场景，强调信息密度和操作效率：
- 主色：深灰蓝 `#1e293b`，专业稳重
- 强调色：亮绿 `#10b981`，用于通过/正常状态；橙红 `#f59e0b`，用于警告/风险
- 按钮：直角矩形，2px 边框，点击反馈明显
- 字体：`"JetBrains Mono"` 等宽字体用于数值显示，`"PingFang SC"` 用于中文内容
- 布局：顶部标签导航 + 卡片/表格内容区，无多余装饰
- 图标：线性简约图标，使用 lucide-react

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI Elements |
|-----------|-------------|-------------|
| 型号选择页 | 型号网格 | 3 列网格布局，型号卡片高亮选中状态，置顶机型带星标 |
| 屏幕方案页 | 方案卡片 | 白色卡片，深灰边框，参数用彩色进度条可视化 |
| 兼容提醒页 | 注意项列表 | 左侧图标 + 标题 + 详情，风险项用橙红色背景高亮 |
| 库存报价页 | 报价表格 | 斑马纹表格，毛利列根据数值颜色渐变 |
| 对比清单页 | 解释卡 | 白色大卡片，左右对比布局，底部推荐按钮 |

### 4.3 响应式

桌面端优先设计，支持平板适配。断点：
- ≥ 1024px：3 列型号网格，完整表格
- 768px - 1023px：2 列型号网格，简化表格列
- < 768px：单列布局，横向滚动表格

### 4.4 交互细节

- 所有按钮/卡片有 hover 状态（边框高亮、轻微阴影）
- 页面切换有淡入过渡（150ms）
- 进度条加载有从左到右动画
- 表格排序点击有箭头方向指示
- 收藏按钮有实心/空心切换动画
